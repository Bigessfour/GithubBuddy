A desktop + web app that teaches Code Platoon students professional terminal and GitHub workflows through guided, executable checklists.

What We’ve Built So Far

📋

Guided Daily Workflows

Select Week 2 Day 4 (or any day when local course content is present) and get a step-by-step checklist with clear explanations of why each action follows GitHub best practices.

🖥️

Real Desktop App (Windows + macOS)

Runs as a native application using Electron + electron-vite. Same codebase works on both platforms with hot reload during development.

📂

Dynamic Course Content

Optionally reads from a local clone of the private Code Platoon repo (aico-echo) to automatically populate available weeks and days.

⚡

Safe Command Execution

Users can actually run the commands from the checklist. Every execution shows a preview + requires confirmation before running inside the chosen workspace folder.
